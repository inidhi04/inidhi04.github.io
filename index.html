github.io page for publishing Data Visualizations

Steps to create a github.io page using GUI:

  • Create your GitHub account
  • On the home page, click on New to create a new respository. Enter the name as username.github.io
  • Add a new file, in this case index.htmlon the Code page of newly created github.io repository. Commit.
  • In the browser, go to username.github.io to see the published contents of index.html
